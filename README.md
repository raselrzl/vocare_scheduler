# vocare-care

A calendar-based caregiving scheduling system built for the Vocare Fullstack Challenge.

## Tech Stack

- Next.js
- TailwindCSS
- shadcn/ui
- Supabase (Database & Auth)
- Google OAuth
- TypeScript
- nylas

## Features Implemented

- [x] Google Authentication
- [x] Supabase database
- [x] Shadcn/ui integrated for UI components
